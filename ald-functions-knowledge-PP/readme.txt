V1.2.1
Un plugin de fonctionnalités :

n°1 : ajouter un texte dans le pied de page du tableau de bord WordPress
n°2 : Charger une feuille de style spécifique et des polices
n°3 : shortcode de mise en page de blocs textes / images
n°4 : éclater le fichier principal en sous-fichiers focalisés sur une seule fonctionnalité
n°5 : internationaliser le plugin (toutes les chaines de caractères affichées peuvent être traduites)
n°6 : charger des templates à partir du plugin ou du thème
